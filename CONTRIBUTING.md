# Contributing

欢迎贡献代码！

## 如何贡献

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的改动 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 开启一个 Pull Request

## 开发环境

```bash
# 克隆项目
git clone https://github.com/yourusername/active-finance-agent.git
cd active-finance-agent

# 安装依赖
make install

# 安装浏览器
make install-browser

# 运行测试
make test
```

## 编码规范

- 使用 4 空格缩进（不要用 tab）
- 遵循 PEP8 编码规范
- 保持代码简洁可读

## 报告Bug

如果你发现了bug，请开一个 [Issue](https://github.com/yourusername/active-finance-agent/issues) 并附上详细的复现步骤。

## 功能请求

欢迎提出新功能请求，也欢迎社区贡献！
